Warenkorbanalyse

Wir knnen mit Market Basket Analysis herausfinden, welche Elemente hufig zusammen gekauft werden. Dazu benutzen wir den Apriori Alghorithmus.

Schritt 1 - DataSet dem UseCase ausrichten

orders_to_products <- orders %>%
  inner_join(products)
Joining, by = "product_id"
orders_products <- orders_to_products %>%
  select(c(orders_id, product_name))

Schritt 2 - Support berechnen

Der Support ist der Anteil eines bestimmtes Produkt an den Gesamtverkufen.

Wir brauchen zuerst die Anzahl der gekauften Produkte insgesamt.

total_products_bought <- orders_products %>%
  count() %>%
  pull(n) %>%
  as.numeric()

Daraus knnen wir den Support pro Produkt berechnen und danach sortieren. Damit haben wir die Reihenfolge der meistverkauften Produkte incl.des Anteils.

t_support <- orders_products %>%
  group_by(product_name) %>%
  summarise(n = n()) %>%
  mutate(support = as.numeric(n)/total_products_bought) %>%
  arrange(desc(support))

Als Threshold fr den Apriori-Algorithmus knnen wir nun den Median des Supports bestimmen.

t_support
support_threshold <- t_support %>%
  summarise(median = median(support)) %>%
  pull(median)

support_threshold
[1] 1.862852e-06

Fr die Warenkorbanalyse bentigen wir die Transaktionen als Key und kommasepariert alle Produkte, welche zu dieser Transaktion gehren. Dies erreichen wir mit folgendem SQL Query:

baskets <- orders_to_products %>% 
  group_by(orders_id) %>%
  summarise(items = as.vector(list(product_name)))
transactions <- as(baskets$items, "transactions")
library("RColorBrewer")
arules::itemFrequencyPlot(transactions,
   topN=30,
   col=brewer.pal(8,'Pastel1'),
   main='Relative Item Frequency Plot',
   type="relative",
   ylab="Item Frequency (Relative)") 

rules <- apriori(transactions, parameter = list(supp = 0.0025, conf = 0.1, target="rules"))
